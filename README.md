# Lotofácil Optimizer PRO

Sistema profissional de otimização de jogos da Lotofácil usando algoritmos matemáticos e inteligência computacional.

## ⚠️ Aviso Importante

Este sistema **NÃO** afirma prever resultados ou aumentar a probabilidade matemática de acerto. O objetivo é puramente **otimizar a distribuição, diversidade e análise** dos jogos.

## 🚀 Funcionalidades

- **Gerador de Jogos**: Geração de jogos válidos da Lotofácil
- **Sistema de Score**: Avaliação em porcentagem (0-100%)
- **4 Algoritmos**: Hill Climbing, Simulated Annealing, Genetic Algorithm, Tabu Search
- **Interface Gráfica**: Moderna com PySide6
- **Exportação**: TXT, Excel, PDF
- **Banco de Dados**: SQLite para histórico

## 📋 Requisitos

- Python 3.12+
- Dependências no requirements.txt
